Before compiling:

- Edit the jirr-dev/Makefile: set SWIG to the path of SWIG 1.3.24 (it must be that version!) on your system, e.g. SWIG=/cygdrive/c/Programme/SWIG-1.3.24/swig.exe -v

- Update SWIG 1.3.24 by the files in jirr-dev/patchSwig/1324, as described in jirr-dev/README.txt


Changes made in Irrlicht:

- CSkyBoxSceneNode.cpp: 1000 units long (instead of 10)
- matrix4.h: added setMatrixData(f32 data[16])
- CCrowdSceneNode.cpp, .h and include/ICrowdSceneNode.h added (with OnRegisterSceneNode() instead of old OnPreRender, and u32 instead of s32 in getMaterialCount() and getMaterial()). Added include to irrlicht.h.
- ISceneManager.h/CSceneManager.h/CSceneManager.cpp: addCrowdSceneNode(...) added.
- COpenGLDriver.cpp: setRenderStates2DMode(): glTexParameteri: GL_LINEAR instead of GL_NEAREST
- IVideoDriver.h/CNullDriver.*/COpenGLDriver.*: draw2DImageRotation(...) (2 versions) added. Implemented only in OpenGL driver.
